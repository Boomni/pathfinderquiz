## Getting Started

### Prerequisites

Before working on the frontend, ensure you have the following installed:

- Node.js and npm

### Installation

If you've cloned the entire Pathfinder Quiz App repository, navigate to the frontend folder:

    cd pathfinderquiz/client

#### Install dependencies:

    npm install

###Project Structure

The frontend code is organized into various directories to maintain a clean and modular structure. Here's an overview:

- `public`: Public assets and HTML template.
- `src`: Source code of the React application.
    - `components`: Reusable UI components.
    - `pages`: Individual pages of the application.
    - `styles`: Stylesheets for styling components.
    - `utils`: Utility functions.

## Usage

To run the frontend development server, use the following command:
    
    npm start

This will start the development server at http://localhost:3000.
